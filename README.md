# Counterfactual Court

Local-first, offline-capable PR review system. Four Gemma 4 agents argue and deliberate; output is a signed, replayable `.verdict` bundle.

**Status:** Phase 0 (foundation). Empty-but-correct scaffolding. CI green on a stub. Models pinned by digest.

For scope, milestones, and the current state of the build, see [`BUILD_PLAN.md`](./BUILD_PLAN.md).

For agent roles, code style, prior art, and architectural decisions, see [`docs/`](./docs).

The full README (architecture diagram, install, demo, benchmarks) is a Phase 3 deliverable. Nothing in this repo is aspirational; everything that exists works.
